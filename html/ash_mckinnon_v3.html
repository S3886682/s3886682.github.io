Platform-agnostic application for automated classification of Australian frog species from environmental audio recordings. Inference runs on-device - no server dependency - built to assist researchers track frog populations over time.

Classifying frog calls currently requires a domain expert to review every recording. FrogFinder eliminates that bottleneck. MFCC feature extraction maps audio signals into a 2D coefficient heatmap, feeding a lightweight on-device ML model.
